SHAKESPEARE'S NEOLOGISMS

About this Website

This website showcases XML data from multiple of Shakespeare's plays to analyze how his use of neologisms (newly coined words or expressions) evolved over time.

By examining the frequency and context of these neologisms, we aim to gain insights into Shakespeare's linguistic creativity and its impact on the English language.

Navigation

All pages are currently under construction, but will include the following features:

"About" page featuring project details and methodology, a "Corpus" page listing Shakespeare's plays in the dataset with metadata, and a "Words" page displaying select neologisms with the old definitions and the modern day ones.

More pages and features will be added as the project progresses and they are deemed neccessary.
